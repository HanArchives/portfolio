Library application

Assignment

For a school assignment I created a visual design for a user interface for searching one or more books from the school library for students. I had to Design in the Amsterdam.nl corporate identity but could adapt it for the target group. The design is made for a tablet in landscape format.

Key Principles

When designing this application, I kept the key principles in mind. All screens have one primary action per screen, are consistent and appearance follows behavior. I only show on the screen what is necessary. I did this so the user won’t be overstimulated. With a strong visual hierarchy, I tried to steer the user the direction I want them to go. I applied the zero state to the wish list. When empty, the user is directed to start searching for books and favorite them.

Design Principles

I designed the feedforward to be small so that the primary action is the dominance. That is why the primary action is bigger. Important elements are dominating the design. The call-to-action buttons and the book covers stand out more than the rest of the design.
