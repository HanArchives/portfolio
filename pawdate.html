Pawdate

Assignment

For a school assignment I had to invent a multidevice application. I designed "Pawdate": an app for dogs and dog owners to meet and go on a Pawdate. Many dogs have little activity or interaction with other dogs. My app ensures that dogs get more interaction with other dogs and more exercise. This makes a dog happier, and a happy dog makes a happy dog owner.

My app allows users to create a profile and specify their preferences. This ensures dogs and their owners find the perfect match for their Pawdate. The map function allows users to find other users and send a Pawdate request. The user is then shown a route to meeting point. When the Pawdate has gone well, the user can send a friend request. The chat function becomes visible when the friend request is accepted. In this way, the user can schedule dates with other users via chat. It is also possible that the user does not want to be visible to others. The “ghost mode” option allows the user to hide from the map.

Progress

First, I interviewed my target audience. I interviewed multiple dog owners to understand what their needs are. I created jobstories that focus on the main functionalities. The outcome functionalities: Map, ghost mode and chat. Five out of ten dog-owners said that they have a hard time trying to reach out to strangers. This is because they don’t know if the other dog owner wants to interact or just simply want to have some alone time with their own dog.

Ghost Mode

This gave me the idea to have a “Ghost mode” option. This gives the user the option to be visible or not visible on the map. The users can decide for themselves if they want to be “online” or “offline”.

Onboarding

While I was interviewing dog owners, it came to my understanding that they also like to find dogs that match with their dog. A lady told me that her small dog was bitten by a bigger dog and ever since, her dog has been afraid of big dogs. This gave me the idea to have an onboarding selection where the user can choose what kind of dog matches with their dog. On the main map function, the user can choose to see all available dogs or only the matching dogs.

Chat

When the user sends a Pawdate request, the other user can accept it or deny it. If it’s accepted the user gets the route to the other user. The Pawdate starts once the user arrives at the destination. If the Pawdate is a success, the user can send a friend request. If the request is accepted, the users can chat with each other and plan another Pawdate.
